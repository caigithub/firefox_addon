# chat-outline

chat-outline adds a thin outline to chat messages.

It runs in Firefox.

It does not send data anywhere.

Install

- From the Add-ons site: click Add to Firefox.
- Or install an XPI file:
	1. Download the XPI.
	2. Open Firefox and go to about:addons.
	3. Click the gear and choose Install Add-on From File.
	4. Pick the XPI and confirm.

Use

- Open a page with chat messages.
- The outline appears automatically.
- To turn it off, go to about:addons and disable the extension.

Troubleshoot

- Reload the page.
- Restart Firefox.
- Some sites block extensions. The outline may not appear.

Uninstall

- Go to about:addons.
- Find chat-outline.
- Click Remove.
